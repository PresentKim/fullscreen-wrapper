# fullscreen.js  
A Light Wrapper for FullScreen API for cross-browser supports  
## Installation  
```sh  
npm install fullscreen.js --save  
yarn add fullscreen.js  
```  
  
  
## Usage  
  
### Javascript  
````javascript  
var fullscreen = require('fullscreen.js');  
fullscreen.require();  
````  
  
### TypeScript  
````typescript  
import fullscreen from 'fullscreen.js';  
fullscreen.require();  
````
